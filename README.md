# GitHub Profile Christmas Kit

Live demo: https://kangchainx.github.io/christmas-readme-assets/

## SVG assets

- Local path: `assets/svg/`
- GitHub Pages base URL: https://kangchainx.github.io/christmas-readme-assets/assets/svg/
- Example: `![christmas-tree](https://kangchainx.github.io/christmas-readme-assets/assets/svg/christmas-tree.svg)`

## Run locally

### Option A: Open the single file

Open `index.html` directly in your browser (requires network access to load CDN assets).

### Option B: Run with Vite

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
